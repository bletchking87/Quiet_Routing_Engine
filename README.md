# Quiet_Routing_Engine
An app for finding the quietest feasible route through a city or otherwise. For those who want to take the scenic route. 


Starting with Barcelona. 
Realised that there is far too much overlap in the Eixample between roads. 
Some roads may be more quiet than others so I have changed from tracking the type-of-road data (with OSMNX) to real reported data with Barcelona with OpenData BCN.
